## 配置 yapi

**域名**

拉取 yapi 接口信息时使用到的域名。

**项目列表**

`名称`

任意名称都行，建议与 yapi 上项目名称保持一致。

`token`

yapi 项目 token。

![](https://fastly.jsdelivr.net/gh/migrate-gitee/img-hosting@latest/2020/11/10/1604992761667.png)

`域名`

未配置则使用之前统一配置的域名。
